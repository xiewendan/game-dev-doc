# 4. 解决方案

## 4.1. 基本方法

* 迭代
  * 和gpt沟通，根据返回的答案，不断修正自己的提问（更清晰、让它想一想），以此来获得更准确的答案
  * 通常一步到位是很难的

* 指导思想--原则
  * 清晰、明确的指令
    * 使用分割符分隔不同给的部分，如`中国在哪`
    * 结构化输出，如json，html格式输出，按照step列出步骤
    * 检查条件满足，如果条件满足，才执行后续的任务
    * 简短的提示，如给出一些成功运行的示例，然后再提问
  * 给模型足够的时间思考
    * 指定完成任务所需的步骤
    * 指示模型在作出结论之前，先推理出自己的解决方案，然后再和已知的方案对比，判断在正确还是错误

* 擅长领域
  * 推理：情感是什么

  * 总结：
    使用最多50个字，做一下总结
